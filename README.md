A modern dental clinic website built with Next.js and Bootstrap for showcasing services, building patient trust, and making appointment booking easier through a clean and responsive interface.

## Overview

Prime Systems Dental is a professional clinic website template focused on:

- Presenting dental services clearly
- Building trust with a modern and polished design
- Supporting mobile, tablet, and desktop users
- Making contact and appointment actions easy to find

## Key Features

- Responsive layout for desktop, tablet, and mobile devices
- Professional homepage for clinic branding and patient confidence
- Dedicated services listing and dynamic service detail pages
- Appointment booking page for patient inquiries
- Gallery page for clinic visuals
- Contact page with clinic communication details
- Reusable component-based structure with Next.js

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Available Scripts

```bash
npm run dev
npm run build
npm run start
npm run sass
```
